###[226]. Invert Binary Tree 
[Easy]
```
New
翻转一棵二叉树。

示例：

输入：

     4
   /   \
  2     7
 / \   / \
1   3 6   9
输出：

     4
   /   \
  7     2
 / \   / \
9   6 3   1


```
>
>
> 递归处理左子树和右子树，如果左右都为null则不用处理。
